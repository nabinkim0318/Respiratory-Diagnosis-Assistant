User/Use Case

The potential users of the application are medical professionals, including doctors, nurses, physicians, and specialists. These professionals will utilize the application to streamline their workflow, access patient information efficiently, and interpret diagnostic results. The system should cater to their specific needs with user-friendly interfaces designed for easy navigation and interpretation of medical data.

Problem Definition

There are several components of this project:

  1. Establish a database system for adult patients, child patients, audio files, and diagnoses.
  2. Develop a machine learning model for the classification of respiratory conditions based on lung sounds.
  3. Convert user input (If it is audio files) to features and use it for classification with metadata(optional).
  4. Return audio features (and original audio if possible) files related to user input if user input is text (disease name).
